# 経理班スケジュール表 起動・運用マニュアル

このシステムは「経理班スケジュール表」をチーム全員で共有・編集するためのWebアプリです。
このマニュアルでは、アプリの動かし方と基本的な使い方を説明します。

---

## 1. アプリの仕組み（重要）

このアプリは「Excelをダブルクリックして開く」ような仕組みではなく、**Webサイトと同じ仕組み**で動いています。

そのため、データを受け渡しする「**バックエンド**」と、画面を表示する「**フロントエンド**」という2つのプログラムを、このサーバー（親PC）上で**常に起動しっぱなしにしておく**必要があります。
もしプログラムが止まったり、黒い画面（PowerShell）を閉じてしまうと、他の人のパソコンからスケジュール表が見られなくなります。

---

## 2. アプリの起動手順

サーバー（このPC）が再起動などでアプリが止まってしまった場合は、以下の手順でシステムを立ち上げてください。

### 手順①：バックエンド（データ処理側）の起動
1. サーバーのデスクトップ等にある `✦ScheduleApp` フォルダを開き、さらに **`backend`** フォルダを開きます。
2. フォルダ内の何もない空間で **Shiftキーを押しながら右クリック** し、「**PowerShell ウィンドウをここで開く**」（または「ターミナルで開く」）を選択します。
3. 黒い（または青い）画面が出たら、以下の文字をコピーして貼り付け、Enterキーを押します。
   ```text
   py main.py --host 0.0.0.0 --port 8000
   ```
4. `Application startup complete.` と表示されたら成功です。**この画面は絶対に閉じず、最小化して裏に隠しておいてください。**

### 手順②：フロントエンド（画面表示側）の起動
1. 再度 `✦ScheduleApp` フォルダを開き、今度は **`frontend`** フォルダを開きます。
2. 先ほどと同じように、**Shiftキー＋右クリック** で「**PowerShell ウィンドウをここで開く**」を選択します。
3. 新しい黒い画面が出たら、以下の文字をコピーして貼り付け、Enterキーを押します。
   ```text
   npm run dev -- --host 0.0.0.0
   ```
4. `http://localhost:3000` のような文字が出たら成功です。**この画面も絶対に閉じず、最小化しておいてください。**

これで起動は完了です！（黒い画面が2つ開いたままになっていれば正解です）

---

## 3. アプリの基本的な使い方

### アクセス方法
各メンバーのパソコンのブラウザ（EdgeやChromeなど）を開き、上部のURLバーに以下のURLを入力します。
（※よく使う方はお気に入りに登録しておくと便利です）
👉 `http://（このサーバーのIPアドレス）:3000/`

### 主な機能
* **📅 予定の登録・編集**
  * 画面上部の「＋ 新規登録」ボタン、またはカレンダーの各日付をクリックすると予定を入力できます。
  * すでに登録されている予定をクリックすると、内容の編集や削除ができます。
* **🏖️ 休暇や出張の登録**
  * 「休暇登録」ボタンから、年休や出張などのステータスを登録できます。
  * 「その他」を選ぶと、行き先などの自由なテキストを入力できます。
* **📝 週間メモ**
  * 画面下部にある「週間メモ」欄は、文字を入力して**1秒待つと自動で保存**されます。「保存ボタン」を押す必要はありません。
* **🔍 人物の絞り込みと印刷**
  * 左上のドロップダウン（「全員表示」と書かれている場所）から特定の人物を選ぶと、その人の予定だけが表示されます。
  * その状態で「🖨 印刷」ボタンを押すと、その人の名前が入った見やすいA4サイズの予定表が印刷できます。
* **⚙️ 人物の追加・並び替え（管理者向け）**
  * 右上の「設定」ボタンから、表示するメンバーの追加や名前の変更、表示順の変更ができます。
  * 表示順の数字が小さい人ほど上に表示されます。
